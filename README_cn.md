## CocosDesktop:

##### CocosDesktop 是 Cocos-BCX 的桌面应用
##### [windows](https://cocosbcx.oss-cn-beijing.aliyuncs.com/CocosDesktop.exe)
##### [mac](https://cocosbcx.oss-cn-beijing.aliyuncs.com/CocosDesktop.dmg)

## node版本:

```
Node.js version 11.+.+.
```

## 安装依赖:

```
1.npm install
```

```
2.npm run dev
```

```
3.npm run build （.exe/dmg 在 `./build`文件夹下）
```
